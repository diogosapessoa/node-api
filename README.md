# Web API with Node + Express + Mongoose

## Course

[Criando APIs com Node](https://balta.io/player/assistir/5c334f86-e717-9a7d-1240-481b00000000)

## Environment values

Add `.env` file with keys and values

```config
NODE_ENV=dev
MONGO_CONNECTION=
JWT_KEY=
```

## Install dependencies

```powershell
npm install
```

## Run

```powershell
npm start
```
